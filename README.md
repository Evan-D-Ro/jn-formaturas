# JN Formaturas

## Informações do projeto

Este projeto é o site/aplicativo da **JN Formaturas**.

## Como editar este projeto

Você pode editar o projeto de algumas maneiras diferentes:

### Usando sua IDE preferida

Para trabalhar localmente, siga os passos abaixo:

```sh
# 1. Clone o repositório
git clone <URL_DO_SEU_REPOSITORIO>

# 2. Navegue até a pasta do projeto
cd <NOME_DO_PROJETO>

# 3. Instale as dependências
npm install

# 4. Inicie o servidor de desenvolvimento com recarregamento automático
npm run dev
